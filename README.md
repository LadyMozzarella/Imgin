#ImgIn
=====

#Installation
=====
Install gem. *once we're on rubygems.org.
```
$ gem install 'imgin'
```

#Usage
=====
Require ImgIn at the top of your file.
```
require 'imgin'
```

Fetch one image URL.
```
Imgin::Image.get( **insert your string** )
```
